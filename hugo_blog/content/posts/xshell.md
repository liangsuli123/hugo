Xshell :一个强大的安全终端模拟软件，它支持SSH1, SSH2, 以及Microsoft Windows 平台的TELNET 协议.

0 帮助 命令 与清屏

help  查看帮助命令

ls   查看ls命令下的具体用法

info  ls 查看ls命令的作用

clear 清屏

1。查看当前路径

pwd （print working directory)

2. 列出当前路径的目录和文件

ls    列出当前目录

ls   *abc 列出文件abc 内的所有文件

ls -l 列出所有的文件

3.查看

cd  进入当前用户主目录（等同于cd ~）

   cd  etl/app/TEMP/abc/ddl

  进入目录etl/app/TEMP/abc/ss

   cd /home 进入home目录

   cd  /       切换到根目录

   cd -         回到上次目录

   cd ..       返回上一级目录

   cd ../..     返回上两级目录


whereis '文件名'   查找
